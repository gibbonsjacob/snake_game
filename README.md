# 🐍 Snake Game (Pygame)

A classic Snake game built using the Python `pygame` library. Simple controls, clean visuals, and easy to run locally.

## 🎮 Features

- Smooth snake movement
- Self-collision and wall detection
- Score tracking
- Restart game on game over
